# Digital Sand Clock
2x 8x8 matrixes positioned diagonaly. Base is 3D printed. <br>
Online simulation available <a href="https://exch.com.ua/e-sand_clock/e-sand_clock.html">here</a>
## Functionalities
- Angle reaction
- Set timer with buttons
- Beeping sound when timer ends
